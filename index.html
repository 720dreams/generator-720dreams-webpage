'Allo, 'allo!

Web app generator is a Yeoman generator for building front-end web apps. It automates common tasks such as JavaScript linting, assets compression, compiling Sass with Autoprefixer etc. so you can get that out of the way and focus on more important stuff. This generator is powered by Gulp, if Grunt is more your cup of tea, check out generator-webapp.

Features

Please check out our gulpfile.js for up to date information on what we support.

  • HTML5 Boilerplate
  • CSS Autoprefixing
  • Built-in preview server with LiveReload
  • Compile Sass
  • Lint your scripts
  • Awesome image optimization
  • Wire-up dependencies installed with Bower
  • Compress and revision your assets

Recipes

If your favorite feature is missing, we have recipes for some common features like:

  • Compile CoffeeScript
  • Use Handlebars for JavaScript templates
  • Compile LESS
  • Use Jade for your markup
  • Host your web app on GitHub Pages
  • Implement asset revisioning
  • Set up React and JSX
